# Snake Game

A classic Snake game built as a static web application using HTML, CSS, and JavaScript.

## How to Run

No build tools are required. Simply open the `index.html` file in your web browser to play the game.

## Controls

### Desktop
- **Arrow Keys** or **WASD**: Move the snake.
- **Spacebar**: Pause or resume the game.
- **R**: Restart the game.

### Mobile
- Use the on-screen touch buttons to control the snake.

## Features

- **Classic Gameplay**: Control a growing snake to eat apples and get the highest score.
- **Increasing Difficulty**: The snake's speed increases every 5 apples eaten.
- **Score Tracking**: Your current score is displayed during the game.
- **High Score**: The highest score achieved is saved in your browser's local storage and displayed.
- **Pause/Resume**: Pause the game at any time.
- **Game Over Screen**: See your final score when the game ends.
- **Responsive Design**: The game scales to fit different screen sizes and includes mobile touch controls.
